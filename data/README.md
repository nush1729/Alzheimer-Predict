# Data Directory

## Raw Data

Place `oasis_longitudinal.csv` in `data/raw/`.

### Download Instructions

**Option 1 — Kaggle (recommended)**
```bash
pip install kaggle
kaggle datasets download -d jboysen/mri-and-alzherimers
unzip mri-and-alzherimers.zip -d data/raw/
```

**Option 2 — Manual**
1. Go to: https://www.kaggle.com/datasets/jboysen/mri-and-alzherimers
2. Download the ZIP
3. Extract `oasis_longitudinal.csv` into `data/raw/`

## Processed Data

`data/processed/` is auto-generated by `python src/preprocess.py`:
- `oasis_clean.csv`     — cleaned dataset
- `top_features.txt`    — selected feature names (one per line)

## Data Dictionary

| Column | Description |
|--------|-------------|
| Subject ID | Patient identifier (removed in preprocessing) |
| MRI ID | MRI session identifier (removed) |
| Group | **Target**: Nondemented=0, Demented=1, Converted=2 |
| Visit | Visit number in longitudinal study |
| M/F | Sex: M=1, F=0 |
| Hand | Handedness (removed — not predictive) |
| Age | Age in years |
| EDUC | Years of education |
| SES | Socioeconomic status (1=highest, 5=lowest) |
| MMSE | Mini-Mental State Examination score (0–30) |
| CDR | Clinical Dementia Rating (0, 0.5, 1, 2) |
| eTIV | Estimated total intracranial volume (mm³) |
| nWBV | Normalized whole brain volume |
| ASF | Atlas scaling factor |
| Delay | Days between visit and MRI (removed) |
